# 个人博客网站

[博客地址](https://blog.pressed.top)

最开始使用的是Jekll，由于Hexo过于优秀所以换成了Hexo，新的博客在本地构建然后发布到gh-pags分支

## master分支

这个分支是Jekyll源代码，已经废弃了，不再使用。

## gh-pages分支

存放博客网站构建后的代码，使用Hexo进行构建，有更多的功能。
**Hexo主题：**[Keep](https://github.com/XPoet/hexo-theme-keep)

- 文章可进行分类、归档
- 文章支持评论
- 博客有Light/Dark模式
- Markdown显示效果更好
